Version: 1.0.4

  • Updated: Minecraft 1.6.2.
  • Updated: Forge 9.10.0.789.
  • Major Change: Removed dependency on JeckelBaseMod. It was only using two JBM classes, so wasn't really necessary.
  • New: Message is now displayed after sign text is erased.
  • New: Message is displayed if sign doesn't have text for a tool to copy or cut.
  • Change: Sign tool durability and text nbt parent tag renamed from JeckelSignMod to signtool. Sign tool text will be lost and durability will be reset.
  • Change: Default sign tool durability increased from 20 to 50.
  • Change: Durability costs for using sign tools has doubled from 1 to 2, with a couple exceptions. Copying text from a sign to a to the copy tool is free. Cutting text from a sign to a cut tool costs 1 and erasing a sign also costs 1. Everything else is 2.
  • New Options: The max durability and the durability cost of using items can now be set in the config file on a per tool basis. For things like the copy and cut tool, different costs can be set for copying text from a sign to the tool, pasting text to from the tool, and clearing the tool's stored text.
  • New Options: A new boolean option is now available to turn durability off for all tools.
  • Removed Options: The tool max durability option has been removed in favor of the new boolean version.
  • Changed Options: The boolean options to allow/disallow individual tool re
  • Misc: Huge overhaul on most of the code. This may bring a few bugs.
  • Removed: Got rid of the mcmod.info file, it became more hassle than it was worth.
